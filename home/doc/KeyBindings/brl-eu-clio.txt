Key Table: EuroBraille AzerBraille, Clio, NoteBraille, PupiBraille, Scriba
==========================================================================

Notes
* The eight dot keys of the braille keyboard are behind the cursor routing keys.
* From left to right, they are: Dot7, Dot3, Dot2, Dot1, Dot4, Dot5, Dot6, Dot8.
* The two keys of the braille keyboard in front of the braille cells, from left to right, are: Backspace, Space.

Default Bindings
----------------

Special Modes
* enter/leave help display: Sharp, H
* enter/leave command learn mode:
  + Sharp, One
  + Sharp, L
* enter/leave preferences menu: H
* enter/leave status display: I
* set display mode attributes/text: Sharp, A
* set screen image frozen/unfrozen: A
* switch to Default Bindings: St
  + ar, Star
  + arp, Sharp

Cursor Functions
* go to screen cursor:
  + Sharp, Up
  + Five
* go back after cursor tracking: Sharp, Down
* bring screen cursor to character: RoutingKey

Vertical Navigation
* go up one line: F
* go down one line: L
* go to beginning of top line:
  + Sharp, Three
  + Sharp, E
  + One
* go to beginning of bottom line:
  + Sharp, Nine
  + Sharp, M
  + Seven
* go up to nearest line with different content: Three
* go down to nearest line with different content: Nine
* go up to previous command prompt: G
* go down to next command prompt: K

Horizontal Navigation
* go backward one braille window: E
* go forward one braille window: M
* go to beginning of line: Sharp, Left
* go to end of line: Sharp, Right

Clipboard Functions
* start new clipboard at character: Star, E, RoutingKey
* append to clipboard from character: Star, F, RoutingKey
* linear copy to character: Star, M, RoutingKey
* rectangular copy to character: Star, K, RoutingKey
* insert clipboard text after screen cursor: Star, L
* search backward for clipboard text: Sharp, G
* search forward for clipboard text: Sharp, K

Configuration Functions
* set track screen cursor on/off:
  + Star, Down
  + Zero
* set screen cursor visibility on/off: Star, G
* set alert tunes on/off: Star, Right
