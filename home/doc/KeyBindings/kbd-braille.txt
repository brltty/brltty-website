Key Table: Bindings for Braille Keyboards
=========================================

Default Bindings
----------------

Special Modes
* enter/leave help display: Dot0+Dot1+Dot2+Dot5
* enter/leave command learn mode: Dot0+Dot1+Dot2+Dot3
* enter/leave preferences menu: Dot0+Dot1+Dot2+Dot3+Dot4
* enter/leave status display: Dot0+Dot2+Dot3+Dot4
* set display mode attributes/text: Dot0+Dot1+Dot4+Dot5
* set display mode attributes: Dot0+Dot1+Dot4+Dot5+Dot8
* set display mode text: Dot0+Dot1+Dot4+Dot5+Dot7
* set screen image frozen/unfrozen: Dot0+Dot1+Dot2+Dot4
* set screen image frozen: Dot0+Dot1+Dot2+Dot4+Dot8
* set screen image unfrozen: Dot0+Dot1+Dot2+Dot4+Dot7
* describe character: Forward+RoutingKey

Cursor Functions
* go to screen cursor: Forward+Select
* go back after cursor tracking: Backward+Select
* bring screen cursor to character: RoutingKey
* bring screen cursor to current line: Dot0+Dot1+Dot2+Dot3+Dot6

Vertical Navigation
* go up one line: Forward+ArrowUp
* go down one line: Forward+ArrowDown
* go to top line: Backward+ArrowUp
* go to bottom line: Backward+ArrowDown

Horizontal Navigation
* go backward one braille window: Forward+ArrowLeft
* go forward one braille window: Forward+ArrowRight
* go backward skipping blank braille windows: Backward
* go forward skipping blank braille windows: Forward
* go to beginning of line: Backward+ArrowLeft
* go to end of line: Backward+ArrowRight
* place left end of braille window at character: Backward+RoutingKey

Clipboard Functions
* start new clipboard at character: ArrowLeft+RoutingKey
* append to clipboard from character: ArrowUp+RoutingKey
* linear copy to character: ArrowRight+RoutingKey
* rectangular copy to character: ArrowDown+RoutingKey
* copy characters to clipboard: RoutingKey+RoutingKey
* insert clipboard text after screen cursor:
  + Backward+Forward
  + Dot0+Dot1+Dot3+Dot4+Dot6
* save clipboard to disk: Dot0+Dot1+Dot3+Dot4+Dot6+Dot8
* restore clipboard from disk: Dot0+Dot1+Dot3+Dot4+Dot6+Dot7

Configuration Functions
* set autorepeat on/off: Dot0+Dot1+Dot2+Dot3+Dot5
* set autorepeat on: Dot0+Dot1+Dot2+Dot3+Dot5+Dot8
* set autorepeat off: Dot0+Dot1+Dot2+Dot3+Dot5+Dot7
* set text style 6-dot: Dot0+Dot2+Dot3+Dot5
* set text style 8-dot: Dot0+Dot2+Dot3+Dot6
* set skipping of lines with identical content on/off: Dot0+Dot2+Dot4
* set skipping of lines with identical content on: Dot0+Dot2+Dot4+Dot8
* set skipping of lines with identical content off: Dot0+Dot2+Dot4+Dot7
* set skipping of blank braille windows on/off: Dot0+Dot1+Dot2
* set skipping of blank braille windows on: Dot0+Dot1+Dot2+Dot8
* set skipping of blank braille windows off: Dot0+Dot1+Dot2+Dot7
* set sliding braille window on/off: Dot0+Dot2+Dot4+Dot5+Dot6
* set sliding braille window on: Dot0+Dot2+Dot4+Dot5+Dot6+Dot8
* set sliding braille window off: Dot0+Dot2+Dot4+Dot5+Dot6+Dot7
* set track screen cursor on/off: Dot0+Dot2+Dot3+Dot4+Dot5
* set track screen cursor on: Dot0+Dot2+Dot3+Dot4+Dot5+Dot8
* set track screen cursor off: Dot0+Dot2+Dot3+Dot4+Dot5+Dot7
* set screen cursor visibility on/off: Dot0+Dot1+Dot4
* set screen cursor visibility on: Dot0+Dot1+Dot4+Dot8
* set screen cursor visibility off: Dot0+Dot1+Dot4+Dot7
* set attribute underlining on/off: Dot0+Dot1+Dot3+Dot6
* set attribute underlining on: Dot0+Dot1+Dot3+Dot6+Dot8
* set attribute underlining off: Dot0+Dot1+Dot3+Dot6+Dot7

Menu Operations
* restore preferences from disk: Dot0+Dot1+Dot2+Dot3+Dot4+Dot7
* save preferences to disk: Dot0+Dot1+Dot2+Dot3+Dot4+Dot8

* go down to next item: Dot4

* go down to last item: Dot5
