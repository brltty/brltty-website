Key Table: Papenmeier BrailleX Live
===================================

Notes
* The long key on the front is the Easy Access Bar.
  It can only be moved one step in each direction (left, right, up, down).
  To emulate the second step, also press any of the lower routing keys
  (those in the row just behind the text cells).
* RoutingKey1 refers to the row of keys just behind the text cells.
* The rockers at each side of the top that don't stay when pressed
  are named the Left and Right Key.

Default Bindings
----------------

Special Modes
* enter/leave help display:
  + RightKeyRear
  + Space+Dot1+Dot2+Dot5
* enter/leave command learn mode:
  + RightKeyFront
  + Space+Dot1+Dot2+Dot3
* enter/leave preferences menu:
  + LeftKeyFront+BarRight1
  + Space+Dot1+Dot2+Dot3+Dot4
* enter/leave status display:
  + LeftKeyFront+BarLeft1
  + Space+Dot2+Dot3+Dot4
* set display mode attributes/text:
  + LeftKeyRear+BarLeft1

* bring screen cursor to character: RoutingKey1
