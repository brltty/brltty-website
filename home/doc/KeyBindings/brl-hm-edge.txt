Key Table: HIMS Braille Edge

Default Bindings
----------------

Special Modes
* enter/leave help display: F1
* enter/leave command learn mode: F2
* enter/leave preferences menu: F4
* enter/leave status display: F3
* set display mode attributes/text: F5
* set screen image frozen/unfrozen: LeftScrollUp+LeftScrollDown+RightScrollUp+RightScrollDown

* go down to next command prompt: RightScrollUp+RightScrollDown+LeftScrollDown
* go up to nearest line with different character: LeftPadLeft+LeftPadUp, RoutingKey

Horizontal Navigation

* go right one braille window: RightScrollDown
