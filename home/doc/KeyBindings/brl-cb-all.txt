Key Table: CombiBraille
=======================

Default Bindings
----------------

Special Modes
* enter/leave help display: Dot1+Dot2+Dot5
* enter/leave preferences menu: Dot1+Dot3+Dot4+Dot6
* enter/leave status display: Dot3+Dot4
* set screen image frozen/unfrozen: Dot1+Dot2+Dot4
* set display mode attributes/text: Dot1+Dot3+Dot4

Cursor Functions
* go to screen cursor: Dot3+Dot6

Vertical Navigation
* go up one line:
  + Thumb2
  + Dot1
* go down one line:
  + Thumb4
  + Dot4
* go to top line:
  + Thumb2+Thumb3
  + Dot1+Dot2+Dot3
* go to bottom line:
  + Thumb3+Thumb4
  + Dot4+Dot5+Dot6
* go to beginning of top line: Thumb1+Thumb2
* go to beginning of bottom line: Thumb4+Thumb5
* go up several lines: Dot1+Dot2
* go down several lines: Dot4+Dot5

Horizontal Navigation
* go left one braille window:
  + Thumb1
  + Dot3
* go right one braille window:
  + Thumb5
  + Dot6
* go to beginning of line:
  + Thumb1+Thumb3
  + Dot2+Dot3
* go to end of line:
  + Thumb3+Thumb5
  + Dot5+Dot6
* go left one character: Dot1+Dot3
* go right one character: Dot4+Dot6
* go left half a braille window:
  + Thumb1+Thumb4
  + Dot2+Dot4+Dot6
* go right half a braille window:
  + Thumb2+Thumb5
  + Dot1+Dot3+Dot5

Clipboard Functions
* start new clipboard at character: Status1, RoutingKey
* insert clipboard text after screen cursor: Dot1+Dot2+Dot3+Dot4

Configuration Functions
* set text style 6-dot/8-dot: Dot2+Dot3+Dot5
* set skipping of lines with identical content on/off: Dot2+Dot4+Dot5
* set sliding braille window on/off: Dot2+Dot4+Dot5+Dot6
* set track screen cursor on/off:
  + Thumb3
  + Dot1+Dot4
* set screen cursor visibility on/off: Dot5
* set alert tunes on/off: Dot2+Dot3+Dot4

Menu Operations
* restore preferences from disk: Dot1+Dot2+Dot3+Dot5
* save preferences to disk: Dot1+Dot2+Dot3+Dot4+Dot5+Dot6

Speech Functions
* stop speaking:
  + Thumb2+Thumb4
  + Dot1+Dot6
* speak current line:
  + Thumb1+Thumb5
  + Dot1+Dot5+Dot6

Clipboard Functions
* append to clipboard from character: Status2, RoutingKey

Clipboard Functions
* linear copy to character: Status3, RoutingKey

Clipboard Functions
* rectangular copy to character: Status4, RoutingKey

Special Modes
* enter/leave help display: Status6
* enter/leave command learn mode: Status5

Cursor Functions
* bring screen cursor to character: RoutingKey
