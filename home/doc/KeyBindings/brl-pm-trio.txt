Key Table: Papenmeier BrailleX Trio
===================================

Notes
* RoutingKey1 refers to any of the keys behind the text cells.

Default Bindings
----------------

Special Modes
* enter/leave help display:
  + RightKeyRear
  + Space+Dot1+Dot2+Dot5
* enter/leave command learn mode:
  + RightKeyFront
  + Space+Dot1+Dot2+Dot3
* enter/leave preferences menu:
  + LeftKeyFront+BarRight1
  + Space+Dot1+Dot2+Dot3+Dot4
* enter/leave status display:
  + LeftKeyFront+BarLeft1
  + Space+Dot2+Dot3+Dot4
* set display mode attributes/text:
  + LeftKeyRear+BarLeft1
  + Space+Dot1+Dot4+Dot5
* set display mode attributes: Space+Dot1+Dot4+Dot5+Dot8
* set display mode text: Space+Dot1+Dot4+Dot5+Dot7
* set screen image frozen/unfrozen:
  + LeftKeyFront+BarDown2
  + Space+Dot1+Dot2+Dot4
* set screen image frozen: Space+Dot1+Dot2+Dot4+Dot8
* set screen image unfrozen: Space+Dot1+Dot2+Dot4+Dot7
* describe character: BarDown2+RoutingKey1

Cursor Functions
* go to screen cursor: LeftKeyFront
* go back after cursor tracking: LeftKeyRear
* bring screen cursor to character: RoutingKey1
* bring screen cursor to current line:
  + LeftKeyRear+BarDown2
  + Space+Dot1+Dot2+Dot3+Dot6

Vertical Navigation
* go up one line: BarUp1
* go down one line: BarDown1
* go to top line: BarUp2
* go to bottom line: BarDown2
* go up to nearest line with less indent than character: BarUp1+RoutingKey1
* go down to nearest line with less indent than character: BarDown1+RoutingKey1

Horizontal Navigation
* go left one braille window: BarLeft1
* go right one braille window: BarRight1
* go to beginning of line: BarLeft2
* go to end of line: BarRight2
* place left end of braille window at character: BarUp2+RoutingKey1

Clipboard Functions
* start new clipboard at character: BarLeft2+RoutingKey1
* append to clipboard from character: BarLeft1+RoutingKey1
* linear copy to character: BarRight1+RoutingKey1
* rectangular copy to character: BarRight2+RoutingKey1
* insert clipboard text after screen cursor:
  + LeftKeyRear+BarDown1
  + Space+Dot1+Dot3+Dot4+Dot6
* save clipboard to disk: Space+Dot1+Dot3+Dot4+Dot6+Dot8
* restore clipboard from disk: Space+Dot1+Dot3+Dot4+Dot6+Dot7

Configuration Functions
* set autorepeat on/off:
  + LeftKeyFront+BarDown1
  + Space+Dot1+Dot2+Dot3+Dot5
* set autorepeat on: Space+Dot1+Dot2+Dot3+Dot5+Dot8
* set autorepeat off: Space+Dot1+Dot2+Dot3+Dot5+Dot7
* set text style 6-dot/8-dot: LeftKeyRear+BarUp1
* set text style 6-dot: Space+Dot2+Dot3+Dot5
* set text style 8-dot: Space+Dot2+Dot3+Dot6
* set skipping of lines with identical content on/off:
  + RightKeyFront+BarLeft1
  + Space+Dot2+Dot4
* set skipping of lines with identical content on: Space+Dot2+Dot4+Dot8
* set skipping of lines with identical content off: Space+Dot2+Dot4+Dot7
* set skipping of blank braille windows on/off:
  + RightKeyFront+BarRight1
  + Space+Dot1+Dot2
* set skipping of blank braille windows on: Space+Dot1+Dot2+Dot8
* set skipping of blank braille windows off: Space+Dot1+Dot2+Dot7
* set sliding braille window on/off:
  + RightKeyFront+BarRight2
  + Space+Dot2+Dot4+Dot5+Dot6
* set sliding braille window on: Space+Dot2+Dot4+Dot5+Dot6+Dot8
* set sliding braille window off: Space+Dot2+Dot4+Dot5+Dot6+Dot7
* set track screen cursor on/off:
  + LeftKeyRear+BarRight1
  + Space+Dot2+Dot3+Dot4+Dot5
* set track screen cursor on: Space+Dot2+Dot3+Dot4+Dot5+Dot8
* set track screen cursor off: Space+Dot2+Dot3+Dot4+Dot5+Dot7
* set screen cursor visibility on/off:
  + LeftKeyRear+BarRight2
  + Space+Dot1+Dot4
* set screen cursor visibility on: Space+Dot1+Dot4+Dot8
* set screen cursor visibility off: Space+Dot1+Dot4+Dot7
* set attribute underlining on/off:
  + LeftKeyRear+BarLeft2
  + Space+Dot1+Dot3+Dot6
* set attribute underlining on: Space+Dot1+Dot3+Dot6+Dot8
* set attribute underlining off: Space+Dot1+Dot3+Dot6+Dot7
* set capital letter blinking on/off: LeftKeyRear+BarUp2
* set alert tunes on/off: RightKeyFront+BarDown1

Menu Operations
* restore preferences from disk:
  + LeftKeyFront+BarLeft2
  + Space+Dot1+Dot2+Dot3+Dot4+Dot7
* save preferences to disk:
  + LeftKeyFront+BarRight2
  + Space+Dot1+Dot2+Dot3+Dot4+Dot8

Speech Functions
* stop speaking: RightKeyRear+BarLeft1
* speak current line: RightKeyRear+BarRight1
* speak from top of screen through current line: RightKeyRear+BarUp1
* speak from current line through bottom of screen: RightKeyRear+BarDown1
* go to current speaking position: RightKeyFront+BarUp1
* decrease speaking volume: RightKeyRear+BarDown2
* increase speaking volume: RightKeyRear+BarUp2
* decrease speaking rate: RightKeyRear+BarLeft2
* increase speaking rate: RightKeyRear+BarRight2
* set autospeak on/off: LeftKeyFront+BarUp1

Keyboard Input
* braille keyboard dot1: Dot1
* braille keyboard dot2: Dot2
* braille keyboard dot3: Dot3
* braille keyboard dot4: Dot4
* braille keyboard dot5: Dot5
* braille keyboard dot6: Dot6
* braille keyboard dot7: Dot7
* braille keyboard dot8: Dot8
* braille keyboard space: LeftSpace
* braille keyboard space: RightSpace
* braille keyboard control: LeftThumb
* braille keyboard meta: RightThumb
* set braille keyboard enabled/disabled: Space+Dot1+Dot3
* set braille keyboard enabled: Space+Dot1+Dot3+Dot8
* set braille keyboard disabled: Space+Dot1+Dot3+Dot7
* enter key: Space+Dot8
* tab key: Space+Dot2+Dot3+Dot5+Dot6
* backspace key: Space+Dot7
* escape key: Space+Dot2+Dot6
* cursor-left key: Space+Dot3
* cursor-right key: Space+Dot6
* cursor-up key: Space+Dot1
* cursor-down key: Space+Dot4
* page-up key: Space+Dot2+Dot3
* page-down key: Space+Dot5+Dot6
* home key: Space+Dot2
* end key: Space+Dot5
* insert key: Space+Dot3+Dot5
* delete key: Space+Dot2+Dot5+Dot6
* switch to previous virtual terminal: Space+Dot1+Dot2+Dot3+Dot6+Dot7
* switch to next virtual terminal: Space+Dot1+Dot2+Dot3+Dot6+Dot8

Special Functions
* restart braille driver: LeftKeyFront+BarUp2
* restart speech driver: RightKeyFront+BarUp2
