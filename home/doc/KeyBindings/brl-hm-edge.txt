Key Table: HIMS Braille Edge
============================

Default Bindings
----------------

Special Modes
* enter/leave help display:
  + F1
  + Space+Dot1+Dot2+Dot5
* enter/leave command learn mode:
  + F2
  + Space+Dot1+Dot2+Dot3
* enter/leave preferences menu:
  + F4
  + Space+Dot1+Dot2+Dot3+Dot4
* enter/leave status display:
  + F3
  + Space+Dot2+Dot3+Dot4
* set display mode attributes/text:
  + F5
  + Space+Dot1+Dot4+Dot5
* set display mode attributes: Space+Dot1+Dot4+Dot5+Dot8
* set display mode text: Space+Dot1+Dot4+Dot5+Dot7
* set screen image frozen/unfrozen:
  + LeftScrollUp+LeftScrollDown+RightScrollUp+RightScrollDown
  + Space+Dot1+Dot2+Dot4
* set screen image frozen: Space+Dot1+Dot2+Dot4+Dot8
* set screen image unfrozen: Space+Dot1+Dot2+Dot4+Dot7
* describe character: RightPadUp+RightPadDown, RoutingKey
* show current date and time: RightScrollUp+RightScrollDown

Cursor Functions
* go to screen cursor: F7
* go back after cursor tracking: F6
* bring screen cursor to character: RoutingKey
* bring screen cursor to current line:
  + LeftPadLeft+LeftPadRight
  + Space+Dot1+Dot2+Dot3+Dot6

Vertical Navigation
* go up one line: LeftScrollUp
* go down one line: LeftScrollDown
* go to top line: LeftScrollUp+RightScrollUp
* go to bottom line: LeftScrollDown+RightScrollDown
* go up to nearest line with different content: RightPadUp
* go down to nearest line with different content: RightPadDown
* go up to nearest line with different highlighting: LeftPadUp
* go down to nearest line with different highlighting: LeftPadDown
* go up to first line of paragraph: LeftScrollUp+LeftScrollDown+RightScrollUp
* go down to first line of next paragraph: LeftScrollUp+LeftScrollDown+RightScrollDown
* go up to previous command prompt: RightScrollUp+RightScrollDown+LeftScrollUp
* go down to next command prompt: RightScrollUp+RightScrollDown+LeftScrollDown
* go up to nearest line with less indent than character: LeftPadRight+LeftPadUp, RoutingKey
* go down to nearest line with less indent than character: LeftPadRight+LeftPadDown, RoutingKey
* go up to nearest line with different character: LeftPadLeft+LeftPadUp, RoutingKey
* go down to nearest line with different character: LeftPadLeft+LeftPadDown, RoutingKey

Horizontal Navigation
* go backward one braille window: RightScrollUp
* go forward one braille window: RightScrollDown
* go backward skipping blank braille windows: RightPadLeft
* go forward skipping blank braille windows: RightPadRight
* go to beginning of line: LeftScrollUp+LeftScrollDown
* place left end of braille window at character: LeftPadUp+LeftPadDown, RoutingKey

Clipboard Functions
* start new clipboard at character: RightPadLeft+RightPadUp, RoutingKey
* append to clipboard from character: RightPadLeft+RightPadDown, RoutingKey
* linear copy to character: RightPadRight+RightPadUp, RoutingKey
* rectangular copy to character: RightPadRight+RightPadDown, RoutingKey
* insert clipboard text after screen cursor:
  + RightPadLeft+RightPadRight
  + Space+Dot1+Dot3+Dot4+Dot6
* search backward for clipboard text: LeftPadLeft
* search forward for clipboard text: LeftPadRight
* save clipboard to disk: Space+Dot1+Dot3+Dot4+Dot6+Dot8
* restore clipboard from disk: Space+Dot1+Dot3+Dot4+Dot6+Dot7

Configuration Functions
* set autorepeat on/off: Space+Dot1+Dot2+Dot3+Dot5
* set autorepeat on: Space+Dot1+Dot2+Dot3+Dot5+Dot8
* set autorepeat off: Space+Dot1+Dot2+Dot3+Dot5+Dot7
* set text style 6-dot:
  + LeftScrollDown+RightScrollUp
  + Space+Dot2+Dot3+Dot5
* set text style 8-dot:
  + LeftScrollUp+RightScrollDown
  + Space+Dot2+Dot3+Dot6
* set skipping of lines with identical content on/off: Space+Dot2+Dot4
* set skipping of lines with identical content on: Space+Dot2+Dot4+Dot8
* set skipping of lines with identical content off: Space+Dot2+Dot4+Dot7
* set skipping of blank braille windows on/off: Space+Dot1+Dot2
* set skipping of blank braille windows on: Space+Dot1+Dot2+Dot8
* set skipping of blank braille windows off: Space+Dot1+Dot2+Dot7
* set sliding braille window on/off: Space+Dot2+Dot4+Dot5+Dot6
* set sliding braille window on: Space+Dot2+Dot4+Dot5+Dot6+Dot8
* set sliding braille window off: Space+Dot2+Dot4+Dot5+Dot6+Dot7
* set track screen cursor on/off:
  + F8
  + Space+Dot2+Dot3+Dot4+Dot5
* set track screen cursor on: Space+Dot2+Dot3+Dot4+Dot5+Dot8
* set track screen cursor off: Space+Dot2+Dot3+Dot4+Dot5+Dot7
* set screen cursor visibility on/off: Space+Dot1+Dot4
* set screen cursor visibility on: Space+Dot1+Dot4+Dot8
* set screen cursor visibility off: Space+Dot1+Dot4+Dot7
* set attribute underlining on/off: Space+Dot1+Dot3+Dot6
* set attribute underlining on: Space+Dot1+Dot3+Dot6+Dot8
* set attribute underlining off: Space+Dot1+Dot3+Dot6+Dot7

Menu Operations
* restore preferences from disk: Space+Dot1+Dot2+Dot3+Dot4+Dot7
* save preferences to disk: Space+Dot1+Dot2+Dot3+Dot4+Dot8

* type braille dots [8]: Space+Dot8

Menu Operations

  + LeftScrollDown+RightScrollDown
  + Dot5

  + RightScrollUp
  + Dot3
