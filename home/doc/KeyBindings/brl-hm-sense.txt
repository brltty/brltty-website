Key Table: HIMS Braille Sense
=============================

Default Bindings
----------------

Special Modes
* enter/leave help display: Space+Dot1+Dot2+Dot5
* enter/leave command learn mode: Space+Dot1+Dot2+Dot3
* enter/leave preferences menu: Space+Dot1+Dot2+Dot3+Dot4
* enter/leave status display: Space+Dot2+Dot3+Dot4
* set display mode attributes/text: Space+Dot1+Dot4+Dot5
* set display mode attributes: Space+Dot1+Dot4+Dot5+Dot8
* set display mode text: Space+Dot1+Dot4+Dot5+Dot7
* set screen image frozen/unfrozen:
  + LeftScrollUp+LeftScrollDown+RightScrollUp+RightScrollDown
  + Space+Dot1+Dot2+Dot4
* set screen image frozen: Space+Dot1+Dot2+Dot4+Dot8
* set screen image unfrozen: Space+Dot1+Dot2+Dot4+Dot7
* describe character: F2+F4, RoutingKey
* show current date and time: RightScrollUp+RightScrollDown

Cursor Functions
* go to screen cursor: F4
* go back after cursor tracking: F1+F4
* bring screen cursor to character: RoutingKey
* bring screen cursor to current line:
  + F2+F3
  + Space+Dot1+Dot2+Dot3+Dot6

Vertical Navigation
* go up one line:
  + F2
  + LeftScrollUp
* go down one line:
  + F3
  + LeftScrollDown
* go to top line: LeftScrollUp+RightScrollUp
* go to bottom line: LeftScrollDown+RightScrollDown
* go to beginning of top line: F1+F2+Backward
* go to beginning of bottom line: F1+F2+Forward
* go up to nearest line with different content: F2+Backward
* go down to nearest line with different content: F2+Forward
* go up to nearest line with different highlighting: F3+Backward
* go down to nearest line with different highlighting: F3+Forward
* go up to first line of paragraph:
  + F4+Backward
  + LeftScrollUp+LeftScrollDown+RightScrollUp
* go down to first line of next paragraph:
  + F4+Forward
  + LeftScrollUp+LeftScrollDown+RightScrollDown
* go up to previous command prompt:
  + F1+Backward
  + RightScrollUp+RightScrollDown+LeftScrollUp
* go down to next command prompt:
  + F1+Forward
  + RightScrollUp+RightScrollDown+LeftScrollDown

Horizontal Navigation
* go backward one braille window:
  + Backward
  + RightScrollUp
* go forward one braille window:
  + Forward
  + RightScrollDown
* go backward skipping blank braille windows: F1+F2
* go forward skipping blank braille windows: F3+F4
* go to beginning of line:
  + Backward+Forward
  + LeftScrollUp+LeftScrollDown
* go left one character: F3+F4+Backward
* go right one character: F3+F4+Forward
* place left end of braille window at character: F1+F3, RoutingKey

Clipboard Functions
* start new clipboard at character: F1+F3+F4, RoutingKey
* append to clipboard from character: F2+F3+F4, RoutingKey
* linear copy to character: F1+F2+F3, RoutingKey
* rectangular copy to character: F1+F2+F4, RoutingKey
* insert clipboard text after screen cursor:
  + F1+F2+F3+F4
  + Space+Dot1+Dot3+Dot4+Dot6
* save clipboard to disk: Space+Dot1+Dot3+Dot4+Dot6+Dot8
* restore clipboard from disk: Space+Dot1+Dot3+Dot4+Dot6+Dot7

Configuration Functions
* set autorepeat on/off: Space+Dot1+Dot2+Dot3+Dot5
* set autorepeat on: Space+Dot1+Dot2+Dot3+Dot5+Dot8
* set autorepeat off: Space+Dot1+Dot2+Dot3+Dot5+Dot7
* set text style 6-dot:
  + LeftScrollDown+RightScrollUp
  + Space+Dot2+Dot3+Dot5
* set text style 8-dot:
  + LeftScrollUp+RightScrollDown
  + Space+Dot2+Dot3+Dot6
* set skipping of lines with identical content on/off: Space+Dot2+Dot4
* set skipping of lines with identical content on: Space+Dot2+Dot4+Dot8
* set skipping of lines with identical content off: Space+Dot2+Dot4+Dot7
* set skipping of blank braille windows on/off: Space+Dot1+Dot2
* set skipping of blank braille windows on: Space+Dot1+Dot2+Dot8
* set skipping of blank braille windows off: Space+Dot1+Dot2+Dot7
* set sliding braille window on/off: Space+Dot2+Dot4+Dot5+Dot6
* set sliding braille window on: Space+Dot2+Dot4+Dot5+Dot6+Dot8
* set sliding braille window off: Space+Dot2+Dot4+Dot5+Dot6+Dot7
* set track screen cursor on/off: Space+Dot2+Dot3+Dot4+Dot5
* set track screen cursor on: Space+Dot2+Dot3+Dot4+Dot5+Dot8
* set track screen cursor off: Space+Dot2+Dot3+Dot4+Dot5+Dot7
* set screen cursor visibility on/off: Space+Dot1+Dot4
* set screen cursor visibility on: Space+Dot1+Dot4+Dot8
* set screen cursor visibility off: Space+Dot1+Dot4+Dot7
* set attribute underlining on/off: Space+Dot1+Dot3+Dot6
* set attribute underlining on: Space+Dot1+Dot3+Dot6+Dot8
* set attribute underlining off: Space+Dot1+Dot3+Dot6+Dot7

Menu Operations
* restore preferences from disk: Space+Dot1+Dot2+Dot3+Dot4+Dot7
* save preferences to disk: Space+Dot1+Dot2+Dot3+Dot4+Dot8

Keyboard Input
* braille keyboard dot1: Dot1
* braille keyboard dot2: Dot2
* braille keyboard dot3: Dot3
* braille keyboard dot4: Dot4
* braille keyboard dot5: Dot5
* braille keyboard dot6: Dot6
* braille keyboard dot7: Dot7
* braille keyboard dot8: Dot8
* braille keyboard space: Space
* braille keyboard meta: F2
* braille keyboard control: F3
* type braille dots [7]: Space+Dot7
* type braille dots [8]: Space+Dot8
* backspace key: Dot7
* escape key: Space+Dot2+Dot6
* cursor-left key: Space+Dot3
* cursor-right key: Space+Dot6
* cursor-up key: Space+Dot1
* cursor-down key: Space+Dot4
* page-up key:
  + Space+Dot2+Dot3
  + Space+Dot1+Dot2+Dot6
* page-down key:
  + Space+Dot5+Dot6
  + Space+Dot3+Dot4+Dot5
* home key:
  + Space+Dot2
  + Space+Dot1+Dot3
* end key:
  + Space+Dot5
  + Space+Dot4+Dot6
* insert key: Space+Dot3+Dot5
* delete key: Space+Dot2+Dot5+Dot6
* function key: Space+RoutingKey
* enter key: Dot8
* tab key: Space+Dot4+Dot5
* switch to the previous virtual terminal: Space+Dot1+Dot2+Dot3+Dot6+Dot7
* switch to the next virtual terminal: Space+Dot1+Dot2+Dot3+Dot6+Dot8
* set braille keyboard enabled/disabled: Space+Dot1+Dot3
* set braille keyboard enabled: Space+Dot1+Dot3+Dot8
* set braille keyboard disabled: Space+Dot1+Dot3+Dot7

Menu Bindings
-------------

Special Modes
* enter/leave preferences menu: Dot8

Horizontal Navigation
* go backward one braille window: LeftScrollUp+LeftScrollDown
* go forward one braille window: RightScrollUp+RightScrollDown

Menu Operations
* go up to previous item:
  + LeftScrollUp
  + Dot1
* go down to next item:
  + LeftScrollDown
  + Dot4
* go up to first item:
  + LeftScrollUp+RightScrollUp
  + Dot2
* go down to last item:
  + LeftScrollDown+RightScrollDown
  + Dot5
* select previous choice:
  + RightScrollUp
  + Dot3
* select next choice:
  + RightScrollDown
  + Dot6
* go to previous menu level: Dot7
