Computer Keyboard Bindings

In order to use one of these keyboard tables, you need to explicitly specify it in one of the following ways:
  • Use BRLTTY's -k (or --keyboard-table=) command line option.
  • Use the keyboard-table configuration file (brltty.conf) directive.
