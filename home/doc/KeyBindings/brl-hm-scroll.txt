Key Table: HIMS Braille Sense with Four Scroll Keys
===================================================

Default Bindings
----------------

Special Modes
* enter/leave help display:
  + Space+Dots125
  + LeftScrollUp+LeftScrollDown+RightScrollUp+RightScrollDown
* enter/leave command learn mode: Space+Dots123
* enter/leave preferences menu:
  + Space+Dots1234
  + RightScrollUp+RightScrollDown+LeftScrollDown
* enter/leave status display:
  + Space+Dots234
  + RightScrollUp+RightScrollDown+LeftScrollUp
* set display mode attributes/text:
  + Space+Dots145
  + LeftScrollDown+RightScrollDown
* set display mode attributes: Space+Dots1458
* set display mode text: Space+Dots1457
* set screen image frozen/unfrozen:
  + Space+Dots124
  + LeftScrollDown+RightScrollUp
* set screen image frozen: Space+Dots1248
* set screen image unfrozen: Space+Dots1247
* describe character: F2+F4, RoutingKey

Cursor Functions
* go to screen cursor or go back after cursor tracking:
  + F4
  + RightScrollUp+RightScrollDown
* bring screen cursor to character: RoutingKey
* bring screen cursor to current line:
  + Space+Dots1236
  + F2+F3

Vertical Navigation
* go up one line:
  + F2
  + LeftScrollUp
* go down one line:
  + F3
  + LeftScrollDown
* go to top line: LeftScrollUp+LeftScrollDown+RightScrollUp
* go to bottom line: LeftScrollUp+LeftScrollDown+RightScrollDown

Horizontal Navigation
* go backward one braille window: RightScrollUp
* go forward one braille window: RightScrollDown
* go backward skipping blank braille windows: F1+F2
* go forward skipping blank braille windows: F3+F4
* go to beginning of line: LeftScrollUp+LeftScrollDown
* place left end of braille window at character: F1+F3, RoutingKey

Clipboard Functions
* start new clipboard at character: F1+F3+F4, RoutingKey
* append to clipboard from character: F2+F3+F4, RoutingKey
* linear copy to character: F1+F2+F3, RoutingKey
* rectangular copy to character: F1+F2+F4, RoutingKey
* insert clipboard text after screen cursor:
  + Space+Dots1346
  + F1+F2+F3+F4
* save clipboard to disk: Space+Dots13468
* restore clipboard from disk: Space+Dots13467
