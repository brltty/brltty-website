Key Table: HandyTech Bookworm
=============================

Default Bindings
----------------

Special Modes
* enter/leave help display: Backward+Forward
* enter/leave preferences menu: Escape+Enter
* set screen image frozen/unfrozen: Backward+Forward+Enter
* set display mode attributes/text: Escape+Forward

Cursor Functions
* go back after cursor tracking: Escape+Backward
* bring screen cursor to character #1: Enter

Vertical Navigation
* go up one line: Enter+Backward
* go down one line: Enter+Forward

Horizontal Navigation
* go left one braille window: Backward
* go right one braille window: Forward
* go to beginning of line: Escape+Enter+Backward
* go to end of line: Escape+Enter+Forward

Configuration Functions
* set track screen cursor on/off: Escape
* set screen cursor style block/underline: Backward+Forward+Escape

Menu Bindings
-------------

Special Modes
* enter/leave preferences menu: Enter

Horizontal Navigation
* go left one braille window: Backward
* go right one braille window: Forward

Menu Operations
* go to previous item: Enter+Backward
* go to next item: Enter+Forward
* go to first item: Escape+Enter+Backward
* go to last item: Escape+Enter+Forward
* select previous choice: Escape+Backward
* select next choice: Escape+Forward
* restore preferences from disk: Escape
* save preferences to disk: Escape+Enter
